# FpsControllerAnimated
A simple fps controller with animations

# Pre-requisites
+ Dotnet sdk (preferrably dotnet 8)
+ Godot 4.3 stable (dotnet version)
+ 5 tomatoes

# Features
The animations : 
- Idle
- Run
- Jump
- Land
- Die
- Camera sway during movement
- Camera shake : continous & one-shot
